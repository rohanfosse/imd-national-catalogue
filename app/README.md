# `app/`

Streamlit dashboard (Phase 4 of the [roadmap](../README.md#roadmap)).

Once `v1.0.0` is published, the app will mirror the sister
`gbfs-audit.streamlit.app` (Paper 01) but at commune resolution:

1. National IMD map with population-strata filter.
2. IES double-penalty layer + sortable list of the 362
   cycling-poverty deserts.
3. M / I / S / T decomposition tab.

Deployment target: `imd-national.streamlit.app`.
