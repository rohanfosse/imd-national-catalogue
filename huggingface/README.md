# `huggingface/`

Staging area for the Hugging Face dataset card and the Croissant
metadata that will accompany the `rohanfosse/imd-national-catalogue`
release.

The card is being drafted in Phase 3 of the
[roadmap](../README.md#roadmap); placeholders here are kept so that
the eventual `huggingface-cli upload` ships from a stable path.
