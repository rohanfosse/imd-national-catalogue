# `experiments/`

Reproductions of the **B14 – B22** experiments that anchor Paper 04.
The upstream working copies live in
`papers/03_imd_bayesian/experiments/` of the `bikeshare-data-explorer`
research repository; this folder will receive cleaned, pinned ports
in Phase 2 of the [roadmap](../README.md#roadmap):

| # | What it validates |
|---|---|
| B14 | Bayesian simplex winning the tournament with INSEE part-vélo |
| B16 | +18 pts R² over the Cerema infrastructure baseline |
| B18 | Out-of-sample ρ = +0.42 in 5-fold CV |
| B20 | National panel on 34,858 communes |
| B21 | National IES — 362 cycling-poverty deserts |
| B22 | Regional decomposition (no political signal) |
